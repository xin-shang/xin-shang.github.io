WEB 程序前后端通信测试 三

前言

通过 WEB 程序前后端通信测试 二 我们完成了初步的后端框架。并用试例代码进行通信尝试已经取得成功了。

本篇文章将对前端的请求代码进行进一步的开发测试。主要完成以下功能：

  • 页面跳转
  • 跳转后请求对应页面的数据信息

准备

完成 WEB 程序前后端通信测试 二

流程

这篇文章内容比较简单，创建 P1.html 和 P2.html :

P1.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>P1 Page</title>
</head>
<body>
    <h1>P1 Page</h1>
    <div id="content"></div>
    <button onclick="goToP2()">跳转到 P2</button>

    <script>
        // 获取并显示 P1.php 的内容
        fetch('http://localhost:8080/Myself_Backend/app/controllers/P1.php')
            .then(response => response.json())
            .then(data => {
                document.getElementById('content').textContent = JSON.stringify(data, null, 2);
            })
            .catch(error => console.error('Error:', error));

        // 跳转到 P2.html
        function goToP2() {
            window.location.href = 'http://localhost:8080/P2.html';
        }
    </script>
</body>
</html>
