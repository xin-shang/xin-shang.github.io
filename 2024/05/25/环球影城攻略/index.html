大阪环球影城 六月一日

提前准备

入园前

门票

确认单

App

入园后

抽选券

马里奥手环（可以提前买，园区内路过第一个，马里奥内部也有卖）（买完绑定到app里）

购买门票

门票 门票已在携程购买，三张成人票，一日游（无快速券，无提前十五分钟入园）。所以要早到。

确认单 购买完成后会发到邮件，确认单中有二维码。次日才能添加绑定。

App 下载App，打开USJ -> 电子号码券 -> 登记入场券。这样绑定门票。

​ 如果同行人要一起绑定，方便后续操作（是一人一绑，一人多绑，多人多绑）

出发准备

抵达园区的交通

第一段：出发步行至 十三站（阪急） 8分钟 550米

  • 西口进入

第二段：十三站（阪急） 到 大阪梅田站（阪急） 4分钟 2站

  • 阪急 神户，宝冢，京都线 大阪梅田方向 2，4，6 号站台

第三段：大阪梅田（阪急） 步行至 大阪站（JR） 7分钟 500米

​

第四段：大阪站（JR） 到 西九条站（JR） 5分钟 3站

  • 大阪环状线 内环 单向 1号站台

第五段：西九条站（JR） 步行至 西九条站（JR梦笑线） 2分钟 150米

第六段：西九条站（JR梦笑线） 到 环球城（JR梦笑线） 5分钟 2站

  • JR梦笑线 樱岛方向 各站（任何一个站都到环球影城）

第七段：步行至大阪环球影城

进入园区

获取抽选券（园区号码入场券，入园保证券）

前置条件

绑定门票 + 进入环球影城 -> 获取马里奥世界和哈利波特世界的抽选券。

获取方法

打开USJ -> 园区入场号码券/抽选券 -> 选择园区 -> 按步骤操作

- 如果人不多，不需要抽选券，会有橘黄色底白字提醒。
- 抢到什么时间的券，就要什么时间去。门口有人检查。
单人通道

最推荐的十一个项目中七个都有单人通道，如果运用得当可以节省一半的排队时间。

如何实时查询项目和单人通道的开放

打开USJ -> 等候时间&表演秀时间表 -> 点击要查看的项目 -> 适用于单人优先乘坐。

排队方法

到项目点，和工作人员说一个人（single），然后等着排队就好。

园区地图

App上有。细节稍后补充。

游玩路线

  • 入园方案一，入园直接进入任天堂园区。这样全部玩完后还能利用整理券再进一次。但是通常火到一进园就到发放整理券的程度。
  • 入园方案二，入园选择进入哈利波特园区。这样可以完全确定玩上第二火爆的园区，但是主动放弃两次进入任天堂园区的机会。在哈利波特园区领任天堂园区的整理券。
  • 入园方案三。入园直接进入最近的园区，这样线路最合理，但实际上并不是最佳选择。（人多不算）。

所以推荐规划：哈利波特园区 | -> 侏罗纪公园 -> 任天堂 | -> 小黄人 -> 纽约 -> 好莱坞 | -> 水世界 -> 小镇亲善村

纪念品

  • 高于 5500 可退税
  • 退税地址在出口左侧（小票原件 & 护照 & 满 5500）
  • 只退当天购物的商品
  • 分布在各项目出口处
  • 最好全部玩完之后再在园区门口买
  • 退税退 10%，退税点收 3.3% 的手续费。实际退税 6.7%

在 USJ - 电子号码券 - 预约乘坐 也有些项目的免费快通。
